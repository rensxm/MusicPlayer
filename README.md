# 🎵 Music Player — Android

Полнофункциональный Android-плеер с тёмной темой, поддержкой жанров, обложек альбомов и плейлистов.

## ⚙️ Шаги по установке в Android Studio

### 1. Открыть проект
- Запусти Android Studio
- **File → Open** → выбери папку `MusicPlayer`
- Дождись синхронизации Gradle (может занять 1-3 мин при первом запуске)

### 2. Проверить SDK
- **File → Project Structure → SDK Location**
- Убедись, что установлен **Android SDK 34** (compileSdk)
- Если нет — **SDK Manager → Android 14.0 (API 34) → Install**

### 3. Проверить Kotlin плагин
- **File → Settings → Plugins → Kotlin** — должен быть версии `1.9.x`

### 4. Синхронизировать Gradle
- Нажми **"Sync Now"** в жёлтой плашке вверху, или
- **File → Sync Project with Gradle Files**

### 5. Запустить
- Подключи Android-устройство (USB-отладка включена) или создай эмулятор API 26+
- Нажми ▶️ **Run**

## ✅ Функции

| Функция | Описание |
|---|---|
| 📂 Автосканирование | При первом запуске находит все MP3/FLAC/AAC на устройстве |
| 🎨 Обложки альбомов | Отображаются везде — в списке, мини-плеере и полном плеере |
| 🎭 Жанры | Читаются из метаданных ID3, отображаются в виде чипа |
| 🔀 Сортировка | По названию, дате добавления, жанру, исполнителю, альбому, году |
| 🔍 Поиск | По названию, исполнителю и альбому в реальном времени |
| 📋 Плейлисты | Создание, переименование, удаление, добавление треков |
| 🎨 Палитра цветов | Полный плеер меняет фон под цвет обложки (Palette API) |
| 🔁 Повтор | Нет / Все / Один трек |
| 🔀 Shuffle | Случайный порядок воспроизведения |
| ⏯ Мини-плеер | Всегда виден в нижней части экрана |
| 🔔 Уведомление | Управление воспроизведением из шторки уведомлений |
| 💾 База данных | Room — треки и плейлисты сохраняются между сессиями |

## 🔧 Возможные ошибки и решения

### ❌ `Unresolved reference: MusicBinder`
**Причина:** KSP/KAPT не завершил обработку.
**Решение:** `Build → Clean Project` → `Build → Rebuild Project`

### ❌ `Manifest merger failed`
**Решение:** Убедись, что `minSdk` в `build.gradle` = **26** или выше.

### ❌ Нет треков после запуска
**Причина:** Не выдано разрешение на доступ к файлам.
**Решение:** Перейди в **Настройки телефона → Приложения → Music Player → Разрешения → Файлы** → включи.

### ❌ `Room schema export` warning
Это просто предупреждение, не ошибка. Можно игнорировать.

### ❌ Gradle синхронизация зависает
**Решение:** Проверь интернет-соединение — Gradle скачивает зависимости (~150 МБ при первом запуске).
